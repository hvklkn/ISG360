# ISG360 Admin Panel UI

Occupational health and safety admin panel frontend built with:

- Next.js App Router
- TypeScript
- Tailwind CSS

Included routes:

- `/login`
- `/dashboard`
- `/companies`
- `/employees`
- `/trainings`
- `/risks`
- `/documents`

Environment:

- Copy `.env.local.example` to `.env.local`
- Set `NEXT_PUBLIC_API_BASE_URL` (default: `http://localhost:4000/api`)

Run locally after installing dependencies:

```bash
npm install
npm run dev
```
